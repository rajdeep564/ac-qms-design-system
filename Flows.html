When you change a page

  1. Update the screen row in the registry above (status, params, role).
  2. Re-run the flow that contains that screen — links are in the tables.
  3. If the screen uses ?docNo or ?batchNo, test invalid-param regression URLs.
  4. If it is a role dashboard, confirm data-dashboard-role and wrong-role redirect.
  5. Push to GitHub; hosted site redeploys from main / master.
